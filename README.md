# Customer Segmentation using Machine Learning

## 📌 Overview
This project uses Machine Learning to group customers based on their purchasing behavior.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## 🚀 Features
- Data preprocessing
- K-Means clustering
- Customer segmentation

## 📊 Output
- Grouped customers into different segments
- Helps in targeted marketing

## 🎯 Result
Improves business decision-making and increases sales efficiency
